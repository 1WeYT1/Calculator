# Архитектура калькулятора

## Как устроена программа

Калькулятор состоит из двух частей:
1. **Математические функции** — считают результаты
2. **Интерфейс** — общается с пользователем

## Все функции калькулятора

### Математические функции
| Функция | Что делает |
|---------|------------|
| `add(a, b)` | Складывает a и b |
| `subtract(a, b)` | Вычитает b из a |
| `multiply(a, b)` | Умножает a на b |
| `divide(a, b)` | Делит a на b (проверяет деление на 0) |
| `power(a, b)` | Возводит a в степень b |
| `mod(a, b)` | Остаток от деления a на b |
| `square_root(a)` | Корень квадратный из a |

### Функция интерфейса
| Функция | Что делает |
|---------|------------|
| `calculate()` | Главная функция, запускает калькулятор |

Как добавить новую операцию
Хотите добавить, например, "процент от числа"?

Добавить функцию в calculator.py:
def percent(a, b):
    """Сколько процентов a составляет от b"""
    if b == 0:
        return "Ошибка: деление на ноль!"
    return (a / b) * 100
Добавить пункт в меню (в функции calculate()):
print("8. Проценты")
Добавить обработку:
elif choice == "8":

# API документация калькулятора

## Описание всех функций

### add(a, b)
Складывает два числа.
- **a** (число) — первое слагаемое
- **b** (число) — второе слагаемое
- **Результат:** сумма a и b

**Пример:** `add(5, 3)` → `8`

---

### subtract(a, b)
Вычитает второе число из первого.
- **a** (число) — уменьшаемое
- **b** (число) — вычитаемое
- **Результат:** разность a и b

**Пример:** `subtract(10, 4)` → `6`

---

### multiply(a, b)
Умножает два числа.
- **a** (число) — первый множитель
- **b** (число) — второй множитель
- **Результат:** произведение a и b

**Пример:** `multiply(6, 7)` → `42`

---

### divide(a, b)
Делит первое число на второе.
- **a** (число) — делимое
- **b** (число) — делитель
- **Результат:** 
  - Если b ≠ 0: частное a / b
  - Если b = 0: "Ошибка: деление на ноль!"

**Примеры:**
- `divide(10, 2)` → `5.0`
- `divide(10, 0)` → `'Ошибка: деление на ноль!'`

---

### power(a, b)
Возводит число в степень.
- **a** (число) — основание
- **b** (число) — показатель степени
- **Результат:** a в степени b

**Пример:** `power(2, 3)` → `8`

---

### mod(a, b)
Возвращает остаток от деления.
- **a** (число) — делимое
- **b** (число) — делитель
- **Результат:**
  - Если b ≠ 0: остаток от деления
  - Если b = 0: "Ошибка: деление на ноль!"

**Примеры:**
- `mod(10, 3)` → `1`
- `mod(10, 0)` → `'Ошибка: деление на ноль!'`

---

### square_root(a)
Вычисляет квадратный корень числа.
- **a** (число) — число для извлечения корня
- **Результат:**
  - Если a ≥ 0: квадратный корень из a
  - Если a < 0: "Ошибка: нельзя извлечь корень из отрицательного числа!"

**Примеры:**
- `square_root(16)` → `4.0`
- `square_root(-4)` → `'Ошибка: нельзя извлечь корень из отрицательного числа!'`
